# mp3joiner-with-ffmpeg-and-windows-11-batch
Batch script that uses ffmpeg to join a folder of mp3s into one giant mp3.
Use case: you downloaded an album of songs as mp3s and want to play them one after another. For whatever reason, you don't want to use a 'play this mp3 after this mp3' program, so you use this batch script to join all your mp3s into one giant mp3 and then you just play that.
Requirements: you need ffmpeg downloaded and set up, each mp3 has to be numbered, and there can't be any special characters in the names of each mp3 file.
